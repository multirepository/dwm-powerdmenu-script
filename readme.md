Installation:
  ```
  cd ~ && git clone https://github.com/multirepository/dwm-powerdmenu-script
  ```
  ```
  sudo cp ~/dwm-powerdmenu-script/powerdmenu /usr/local/bin
  ```
  ```
  sudo chmod +x /usr/local/bin/powerdmenu
  ```
Dependencies: ```dmenu```, ```systemd```

Usage:
open dmenu ```Mod + P```, search ```powerdmenu```

Colors note for my dwm config:
```-nb "#222222" -nf "#bbbbbb" -sb "#595856" -sf "#eeeeee"```

Have a good day!
